# MOS wsdd-native

mos-wsdd-native provides **prebuilt and packaged wsdd-native binaries** for use within the MOS ecosystem.

This repository contains the **build scripts, packaging logic, and configuration files** required to compile and package wsdd-native for MOS.

---

## Overview

wsdd-native is built from upstream sources and packaged for integration into MOS.

No functional changes to wsdd-native itself are intended.  
The goal of this repository is to provide **consistent, reproducible wsdd-native packages** for MOS installations.
